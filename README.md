# Chernarus Pristine Loot

This repo contains the DayZ Chernarus 1.28 configuration files for pristine loot. All loot spawns in full and in pristine condition. In addition, all weapons spawn with full magazines, optics, and a chance for lights and suppressors where appropriate. In order to adjust the chance for attachments, edit the presets in cfgrandompresets.xml.
